About this Site

This site serves a dual purpose. First, it gives me a site where I can put up some of my personal ramblings. Second, (and arguably more importantly) the site itself serves as an example of my skill. It's an AngularJS, Bootstrap 3, and Node.js website. On the server side, it features RESTful endpoints (powered by Express), and an in-process database I've written and maintain, called trivialdb.

If you're curious, I more than welcome an inspection of the source code. Check out the project on GitHub. Feel free to commit any fixes you find, or to use the code in your own project; I've licensed it as MIT, which should let you do nearly anything you might want to with the code.
